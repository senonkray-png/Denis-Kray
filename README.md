## 🌟 Overview

This portfolio is a bespoke scrollytelling experience crafted to highlight motion, interactive storytelling, and a versatile tech stack. It blends modern web standards with high-fidelity visualization techniques.

## 🛠️ Tech Stack

- **Core**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animation**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Flip), [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Vanilla CSS for custom components)
- **Smooth Scroll**: [Lenis](https://github.com/darkroomengineering/lenis)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Real-Time 3D**: [High-fidelity renders & Unreal Engine 5 concepts]

## ✨ Key Features

- **Kinetic Typography**: Large-scale bold typography that reacts to scroll and intent.
- **Horizontal Project Scroll**: A premium, non-linear project showcase using GSAP pinning.
- **Typographic Skills Matrix**: A clean, grid-less expertise list with animated reveals.
- **Side & Corner HUD Animations**: Decorative, cinematic HUD elements (scanlines, orbiters, brackets) that fill the viewport for a high-tech "Unreal" feel.
- **Fluid Custom Cursor**: A custom gravitational cursor effect for enhanced interaction.
- **Premium Palette**: A sophisticated "Cream, Gold, and Grey" theme for a high-end editorial look.

## 📂 Project Structure

- `src/sections/`: Modular components for Hero, About, Skills, Projects, Journey, and Contact.
- `src/components/`: Reusable UI elements like Navbar, Custom Cursor, and 3D scenes.
- `src/hooks/`: Custom React hooks for Lenis and motion management.
- `src/index.css`: Centralized theme tokens and utility classes.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Huzaifa-12Imran/Huzaifa.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```
